# MyAnimeListScraper
A tool that scrapes top 50 anime based on the category
The input parameters permitted are 
    1.all-anime
    2.airing
    3.upcoming
    4.tv
    5.movie
    6.ova
    7.ona
    8.special
    9.bypopularity
    10.favorite

The tool returns an output of the top 50 anime in the format "ranking(an integer)" "anime name"

Main Dependencies: BeautifulSoup
